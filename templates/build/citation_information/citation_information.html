Citation Information

Creating a bundle in PDS4 is a complicated process and those working to compile the bundle should be credited for their work. In the Citation Information section you can input all your authors and editors. Authors contribute to the creation of the files and data within the bundle. If the author is a group of people from a set organization, you should add an 'authors organization'; otherwise the author's organization (and the editors organization) should be set to '0'. You can put in the number of authors and editors on this screen; each author's identification information will be added on the next page. Publication year should be the expected year of release of this bundle. Description can contain a few sentences describing your project (should be concise). You can add keywords for your project that could help in determining search facets (input as a comma separated list: e.g., item1, item2, etc.).
